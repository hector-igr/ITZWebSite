Link Adaptive Components

Coloca a partir de una hoja de cálculo las familias adaptativas indicadas con los puntos indicados y escribe dentro de la misma hoja el GUI de cada elemento colocado.



  • Al finalizar la colocación en la hoja de cálculo se anexara como primera columna el encabezado GUID el cual contendrá el id único de cada elemento correctamente colocado, si hubiese un renglón en el cual no existiese la familia dentro del documento el app colocara un “null” como valor.
  • Si queremos actualizar la posición de cualquiera de los elementos o colocar nuevos solo se anexan los renglones necesarios con la información corregida o aumentada, la aplicación corregirá los elementos que hayan sufrido una transformación y anexara un GUID por cada nuevo elemento colocado.

Requisitos

  • El archivo de Excel tiene que estar cerrado antes de ejecutar la aplicación.
  • El archivo tiene que contener una columna con FamilyName y FamiliType, los puntos tienen que contener como encabezado pt1, pt2, pt3, pt4, etc. Cada campo de coordenadas tiene que estar delimitado por los caracteres “[]”.
