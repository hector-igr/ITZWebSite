Utilidades de Excel

Exporta la informacion de uno o varios modelos rvt por categorias a un archivo de excel divididos en pestañas


Select On Revit


En la vista actual selecciona y separa visualmente los objetos seleccionados en excel. Notas:

  • Se necesita que la hoja de excel contenga guid o id valido.
  • Los elementos a seleccionar tienen que estar dentro de la vista actual.
  • No funciona para elementos en Revit Links.

Update on Revit


Actualiza los parámetros seleccionados en las celdas dentro de el documento actual de Revit. Notas:
  • El parámetro a actualizar tiene que ser de Escritura.
  • El elemento seleccionado tiene que estar dentro de la sección actual de Revit.
  • No opera en elementos en Revit Links.

Override Color On Revit


Genera un "Override" de hatch solido dentro de Revit con el color seleccionado desde Excel Notas: * Excel tiene que contener una columna con encabezado "Color". * Soporta Formatos Condicionales de Excel.

Set Code On Revit

Utilidad para hacer mas eficiente el copiado del código de una tabla a objectos de Revit. Nota:

  • El código a transferir tiene que tener como nombre "Partida" o "Code".
