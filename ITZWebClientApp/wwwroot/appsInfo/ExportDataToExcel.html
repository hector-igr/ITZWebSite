Export Data To Excel

Exporta la informacion de uno o varios modelos rvt por categorias a un archivo de excel divididos en pestañas




Requisitos

  • Esta app funciona con Excel 2016

  • Excel y el archivo a escribir tienen que estar cerrados

  • Si se desea utilizar un archivo existente es importante que el nombre de las hojas concuerden con las categorías de revit, y cada hoja debe de existir la celda id como columna inicial y las columnas file y status.

  • La tabla con los archivos debe de estar delimitada por una columna vacía al lado derecho y un fila vacía al final de el ultimo registro.

  • Para archivos en BIM 360 solo puede exportar el archivo abierto.

Procedimiento
Excel File Selecciona el archivo de Excel a utilizar, checar los requisitos del archivo.
Parameters: Categories Categorías a exportar.
Parameters: Properties of Type Parámetros de Tipo.
Parameters: Properties of Instance Parámetros de Instancia.
Parameters: Custom Si el parámetro no esta en los listados de instancia o tipo escribirlos parámetros separados con el caracter ",".
Delete Lost Records En caso de exportar a un archivo existente la app borrara las filas cuyo Id no se encuentren en el modelo(s).
Export Only Visible Exporta solo los elementos presentes en la vista actual.
Documents Listado de documentos rvt a exportar. Por default se enumera el documento actual y si este tuviese archivos enlazados (links) aparecerá su ruta.
Add Documents Agregar un documento para exportar.
