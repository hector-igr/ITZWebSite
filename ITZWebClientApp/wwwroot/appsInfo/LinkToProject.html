Link to Project

Conjunto de programas para vincular fechas y avances bidireccionales entre Revit y MS Project.


Link To Project

Escribe o sobre escribe los valores de la linea base actual a parámetros de solo lectura en Revit.

Parámetro Descripción
MSProject_File Nombre del Archivo de Project
AC_TaskName Nombre de la Tarea de project
AC_BL_Start Linea Base Empezar Tarea
AC_BL_Finish Linea Base Finalizar Tarea
AC_TaskActive Tarea Activa

Unlink Project

Elimina la vinculación con Project

Calculate Status

Actualiza el estado de todos los objetos con respecto a la linea base ya importada de project con respecto a las fechas y avances actuales Al vincular las tareas el programa asigna diferentes "Estados" de tarea los cuales pueden ser utilizados en filtros.

Parámetro Descripción
Accomplished Tarea Terminada
On Time Tarea en tiempo
Behind Tarea por abajo de lo planeado
Early Tarea adelantada
Overdue Tarea ya debió de ser empezada y no ha sido iniciada
Uninitiated Tarea no iniciada y sin atrasos

Data Input

Utilidad para escribir fechas y % de avance en los campos AC_ActualStart, AC_ActualFinish y AC_Advance. Cada vez que se mete algún valor automáticamente se actualiza su "estatus" con respecto a la fecha de corte.


Refresh From Project

Si la linea base se mueve dentro de project se puede actualizar las fechas y volver a hacer el calculo del estatus la actualización, no mueve las fechas actuales ni el porcentaje de avance.


Select From Project

Muestra los objetos relacionados con la tarea de project en el documento actual de Revit.


La app soporta la posición jerárquica, en este ejemplo la app selecciona todas las subtareas.


Export Advance To Project

Exporta el avance del documento actual en Revit (con los links activos) al documento activo en Project. Para que exporte necesita los siguientes campos siguientes en MS Project :

Nombre de Campo Unidad de Campo Comentarios
Total Quantity (BIM) Numero Exporta la cantidad total encontrada en el archivo tomando como base el "Parameter Extraction (BIM)"
Advance Amount (BIM) Numero Exporta el avance en cantidades tomando como base el "Parameter Extraction (BIM)"
Parameter Extraction (BIM) Texto Parámetros de extracción, por default la aplicación soporta los parámetros "Area", "Longitud", "Pieza" y "Volumen" si se desea extraer otro dato se necesita colocar correctamente el nombre del parámetro.
Units (BIM) Texto Las Unidades son importadas con respecto a las actuales en Revit.
