Add Bars to Segment

Genera varillas a lo largo del segmento del estribo seleccionado.



Procedimiento

Method Distance or Offset
Ends of Set La varilla es colocada con respecto a longitud total del set mas el offset indicado.
Solid Projection La varilla es colocada con respecto a la proyección del solido que hospeda el estribo mas el offset indicado.
Length and Midpoint La varilla es colocada con respecto a el midpoint del set mas la longitud indicada.

Notas

  • Al seleccionar un segmento curvo solo coloca una varilla
  • La utilidad esta pensada para colocar rápidamente varillas longitudinales, para cambiar la orientación o el numero de varillas utilizar las herramientas de revit normales.
