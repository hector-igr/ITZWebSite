Procedimiento

Method Distance or Offset
Ends of Set La varilla es colocada con respecto a longitud total del set mas el offset indicado.
Solid Projection La varilla es colocada con respecto a la proyecion del solido que hospeda el estribo mas el offset indicado.
Length and Midpoint La varilla es colocada con respecto a el midpoint del set mas la longitud indicada

Notas

  • Al seleccionar un segmento curvo solo coloca una varilla
  • La utlidad esta pensada para colocar rapidamente varillas longitudinales, para cambiar la orientacion o el numero de varillas utilzar las herramientas de revit normales.
