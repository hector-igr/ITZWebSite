Conceptos

Rebar ModelAgrupacion de estribos que comparten una misma distribucion dentro de un Path
PathLinea o conjunto de lineas que definen la trayectoria de todos los Stirrup Groups dentro de un Rebar Model
Stirrup GroupEstribo o conjunto de estribos que definen la forma de todos los subsecuentes dentro del set
MembersEstribos miembros de un Stirrup Group
RuleRegla con el cual los Stirrup Groups seran distribuidos a lo largo del Path
Rebar Model
AddAgrega un Nuevo Rebar Model al documento
DeleteBorra el Rebar Model seleccionado del documento
Path
Add Agrega un nuevo path al Rebar Model. Restringe la seleccion a Curve Models o un grupo que contenga exclusivamente Curve Models y que esten conectados en sus extremos.
RemoveRemueve el path de el Rebar Model. No elimina el elemento
Rule
EditEditar la regla del path seleccionado
Stirrup Group
AddAgrega un nuevo Stirrup Group al Rebar Model Seleccionado
DeleteBorra el Stirrup Group del Rebar Model Seleccionado
Members
AddAgregar estribos al Rebar Group
RemoveRemueve el estribo del Rebar Group. No elimina el elemento
Los estribos miembros tienen que estar orientados al sentido del path
Rebar
BuildConstruye el Rebar Model con respecto a los parametros establecidos en su Path asi como en sus miembros
