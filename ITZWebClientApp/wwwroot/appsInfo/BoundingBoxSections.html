Notas

  • Checar que el parametro Elemento de tipo texto e instancia este dentro de las categorias Structural Rebar, Generic Model y Structural Framing

  • El nombre de los cortes depende del nombre del tipo del elemento + ID del elemento + Tipo de Vista
