Vistas por Elemento

Genera vistas frontal, lateral y en planta. La aplicacion esta pensada para facilitar la orientacion de los cortes cuando el objeto no esta en posicion ortogonal



Notas

  • Checar que el parámetro Elemento de tipo texto e instancia este dentro de las categorías Structural Rebar, Generic Model y Structural Framing
  • A cada vista se le asignara un filtro único por utilizando el parámetro Elemento, para poder destacar los objetos involucrados con dicho elemento como pueden ser acero de refuerzo, placas, etc.
  • El nombre de los cortes depende del nombre del tipo del elemento + ID del elemento + Tipo de Vista.
