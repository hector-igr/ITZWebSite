Procedimiento

  • Dentro de configuración en la ficha cabezales mapear los siguientes parámetros

    Cabezal General Type para apoyos intermedios (pilas)
    Cabezal Entrada Type para el primer apoyo (caballete)
    Cabezal Final Type para el ultimo apoyo (caballete)
    Nombre de Apoyo Eje de apoyo sobre el que esta el cabezal
    Esviaje Izquierdo y Derecho Esviaje en las laterales correspondientes del cabezal
    Altura Mínima de Bancos Altura Mínima de bancos
La familia generica tiene que contener en la etiqueta Type Comments Corona para que permita ser seleccionada.
Si algun paño lateral esta inviertido restar a 180° el valor.

La aplicacion coloca el cabezal a la altura minima indicada
