Procedimiento

  • Select Profile. Seleccionar la familia de perfil.
  • Use Type Name for Parameter Prefix. Utiliza el nombre del tipo como prefijo para el nuevo parametro
  • Import as Instance. Importa los parametros como Instancia en el caso opuesto importa los parametros como types.

Notas

  • Solo importa parametros creados por el usuario y que esten dentro de la ficha geometria.
  • No importa parametros que esten relacionados a alguna formula
