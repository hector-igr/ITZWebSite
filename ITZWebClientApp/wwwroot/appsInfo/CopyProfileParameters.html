Copy Profile Parameters

Copia y relaciona los parametros geometricos de una familia 'Profile' al documento activo.



Procedimiento

  • Select Profile. Seleccionar la familia de perfil.
  • Use Type Name for Parameter Prefix. Utiliza el nombre del tipo como prefijo para el nuevo parámetro.
  • Import as Instance. Importa los parámetros como Instancia en el caso opuesto importa los parámetros como types.

Notas

  • Solo importa parámetros creados por el usuario y que estén dentro de la ficha geometría.
  • No importa parámetros que estén relacionados a alguna formula.
