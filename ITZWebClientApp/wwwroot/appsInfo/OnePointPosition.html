One Point Position

Coloca Familias de un punto de inserción utilizando un archivo de Excel que contenga como encabezados "Position X", "Position Y", "Position Z", "Rotation", "FamilyName" y "FamilyType".



Automáticamente busca una columna con nombre "ID" si no la encuentra recorre todas las columnas y genera la columna con los ids de todas las instancias colocadas, si se reimporta las coordenadas los objectos que tengan un ID valido cambiaran de posición sin generar instancias repetidas.

Notas

Las unidades de entrada tienen que ser constantes con las unidades del proyecto
