Procedimiento

  • Dentro de configuración en la ficha cabezales mapear los siguientes parámetros:

    Cimentación Inicio Type para el primer apoyo (caballete)
    Cimentación Regular Type para apoyos intermedios (pilas)
    Cimentación Fin Type para el ultimo apoyo (caballete)
La familia genérica tiene que contener la palabra Apoyo dentro de Type Comments para que permita ser seleccionada.
