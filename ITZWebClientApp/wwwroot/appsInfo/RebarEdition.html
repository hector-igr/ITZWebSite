Contenedores de Varilla

Crea un grupo de varillas que a diferencia de Rebar Set puede contener formas con diferentes dimensiones.

El principal beneficio de convertir a un contenedor es tener una base de datos mas ligera ya que por cada varilla sencilla se genera un registro, el contenedor puede tener un gran numero de varillas con solo un registro

La aplicación separa las varillas seleccionadas en formas y diámetros antes de agruparlos.Los parámetros que son iguales en todas las varillas del contenedor quedan igual, en el caso de que hubiera variaciones se queda el valor vacío

Parámetros de interés:

Quantity Total de varillas
Total Bar Length La sumatoria de Bar Length de todos los estribos
Reinforcement Volume Volumetria aproximada de todos los estribos

Alargar Estribos Beta

Alarga o disminuye estribos en una dirección


La opción mas estable es la Horizontal. Falta desarrollo para las demás opciones.

Explotar Set

Genera varillas individuales de un RebarSet o RebarContainer



Editar Set

Permite cambiar el Type, el gancho inicial-final y invertir ganchos en Rebar Container
