Configuracion de Estribos

Aplicacion que permite automatizar la colocacion de estribos.


Conceptos
Rebar Model Agrupación de estribos que comparten una misma distribución dentro de un Path
Path Linea o conjunto de lineas que definen la trayectoria de todos los Stirrup Groups dentro de un Rebar Model
Stirrup Group Estribo o conjunto de estribos que definen la forma de todos los subsecuentes dentro del set.
Members Estribos miembros de un Stirrup Group.
Rule Regla con el cual los Stirrup Groups seran distribuidos a lo largo del Path.
Rebar Model
Add Agrega un Nuevo Rebar Model al documento
Delete Borra el Rebar Model seleccionado del documento
Path
Add Agrega un nuevo path al Rebar Model. Restringe la selección a Curve Models o un grupo que contenga exclusivamente Curve Models y que estén conectados en sus extremos.
Remove Remueve el path de el Rebar Model. No elimina el elemento.
Rule
Edit Editar la regla del path seleccionado.
Stirrup Group
Add Agrega un nuevo Stirrup Group al Rebar Model Seleccionado.
Delete Borra el Stirrup Group del Rebar Model Seleccionado
Members
Add Agregar estribos al Rebar Group.
Remove Remueve el estribo del Rebar Group. No elimina el elemento.
Los estribos miembros tienen que estar orientados al sentido del path
Rebar
Build Construye el Rebar Model con respecto a los parametros establecidos en su Path así como en sus miembros.
