Procedimiento

  • Dentro de configuración en la ficha Tope Sísmico mapear los siguientes parámetros:

    Neopreno Type para neopreno hospedado por cara

La familia del neopreno tiene que contener en la etiqueta Type Comments Neopreno para que permita ser seleccionada.

La familia del tope sísmico tiene que contener en la etiqueta Type Comments Tope Sísmico para que permita ser seleccionada.
