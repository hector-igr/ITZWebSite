Copiar Varillas

Copia todas las varillas o las seleccionadas de un objeto a otro. Esta pensada para objetos que son iguales geometricamente pero colocados en diferente posiciones.(ej trabe con misma longitud pero con diferente pendiente). o objetos que se edito manualmente algunos estribos y que se quieren propagar en otros elementos.



Procedimiento

  • Seleccionar el host de referencia de donde se copiaran las varillas.
  • Seleccionar las varillas a copiar o oprimir Finish para seleccionar todas.
  • Seleccionar donde se pegaran las varillas.
